VS2019 实用设置

发表于 2019-03-29 | 更新于 2019-03-31 | 分类于 VStudio

本文记录了 VS2019 预览版使用过程中的一些设置，这些设置也同样适用于 VS2017，我们可以根据个人的实际情况进行修改。

滚动条(Scroll Bar)

将滚动条设置为 map mode 后，则该区域会显示当前整个文件的缩略示图，能够以不同的颜色显示当前文件中的修改状态、错误、警告等信息，便于对当前文件有个全局的认识。在 VS2010 中需要使用 Productivity Power Tools 插件中的 Enhanced Scroll Bar 功能，在 VS2019 中已经集成了该功能。
